Часть 1. 
Должен быть реализован код, который генерирует ошибки в тексте.
Функция генерации ошибок должна удовлетворять следующим требованиям.
1. Все сгенерированные ошибки должны фиксироваться в отдельном файле.
2. Для генерации ошибок нужно использовать исходный текст, dict1 - словарь для исходного текста, dict2 - словарь, который включает в себя реально существуеющие слова, которых нет в dict1
3. В качестве параметров нужно использовать n1 и n2, где n1 и n2 - количество слов, до которых редакторское расстояние равно 1 и 2 соответственно.
4. В качестве параметров нужно использовать nspace, т.е. количество ошибок, связанных с отсутствием пробела
5. В качестве параметров нужно использовать ndict2, т.е. количество слов, которые являются ошибка по словарю dict1, но не являются ошибками по словарю dict2, и до которых редакторское расстояние не превышает 2.
6. Ошибки должны создаваться путём пропуска букв, вставки букв, замены букв на другие (лучше на буквы, которые на клавиатуре находятся по соседству), перестановки соседних букв внутри слова
7. Текст с ошибками должен сохраняться в отдельном файле.
